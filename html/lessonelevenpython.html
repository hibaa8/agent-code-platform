Python Array

Arrays are data abstractions that are used to store items. They are very similiar to lists and share many methods and properties, except they are more geared towards arithmetic computations. Furthermore, they come in handy when you want to perform similiar operations on an entire list of items. One way to define an array is to import numpy and use the array() function to define an array. Since cost was an array, we were able to easily every item in cost by 2 simply by dividing the array itself by two. However, if cost was a list, then we would get a type error since python doesn't allow users to perform mathematical operations between data abstractions and numbers.

Another key difference between lists and arrays is that arrays are usually used as containers for elements of the same data types but lists can have different data types. But aside from these two key differences, arrays and lists are very similiar so lists are often used as arrays. But if you want to work with python arrays, you will have to import a library like the array or numpy library.

To define arrays we will use the array library. Simply import array and then do array.array(). array() takes in two parameters, a unicode that specifies the data type of the items in the array and a second parameter which is a list of all the items that will be passed into the array. You can access values in the array using indexing since arrays are 0-indexed. You can modify values in the array by using the format array_name[index] = new_value to change the value corresponding to that array. Use the len() function to find the length of an array. Arrays are iterable, so you can loop through them to access and change items in the array. Below are some functions you can use on arrays. These methods may sound familiar since they can also be used on lists.

append() - adds an element to the end of a list.
copy() - returns a copy of the list.
count() - takes in a parameter x and returns the number of elements with the value x.
clear() - removes all the elements from the list.
extend() - takes in multiple values and adds them to the end of the list.
index() - takes in an item and returns the index of the first item with that specific value.
insert() - takes in an index, value pair. Adds the specified value to the appropriate index in the list.
pop() - takes in an optional index and removes the value at that index in the array. If no index is specified, then it removes the last array item.
remove() - takes in the name of an item and removes the first occurance of that item in the array.
sort() - sorts the list alphabetically or from least to greatest by default. To reverse these defaults, you can do sort(reverse=True).

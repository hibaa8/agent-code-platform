Python Classes/Inheritance

Classes

Object-oriented programming (OOP) is a method of structuring a program by bundling related properties and methods into individual objects. Basically, objects are like components of a system. Think of classes as representing any common object or being like a human. So, lets create a class called 'Human'. First, we need to think of properties that define humans. Some properties include height, weight, and hobbies. Second, what behaviors can humans perform? Well, humans can run, jump, kick, and do so much more but we'll stick with those three for now. Now, we have defined the properties and behaviors that define humans so we can create many human objects that have the properties that we just mentioned. Using this basic example, you can understand class properties, methods, and objects. This class is also visualized in the graphic above.

Inheritance

Now lets say we want to get some properties from another class. But since we are good programmers, we wont just copy paste the methods and properties we need into our new class. Instead, we will inherit the the other class and then take methods and properties that we need.

Parent class - the class being inherited from, also known as the base class

Child class - the class that inherits from the parent class, also called the derived class

The child class gets all the properties of methods of the parent class, as shown below.

You can also add new properties to the child class and build upon the properties that it inherited from the parent class by using an init method. You can also overwrite the values of the instance variables in the parent class by redefining them in the child class. However, if you create an init method, remember to initialize the parent class inside first with all the necessary parameters. If you don't initialize the parent class then you lose access to its properties and methods. To initialize the parent class, you can either call parent_class_name.__init__(self, parameters) or you can do super.__init__(self, parameters).

Similiarly, you can add new methods to the child class and you can overwrite the methods in the parent class by redefining them. Keep in mind that you can use the instance variables and methods of the parent class in your methods. Furthermore, remmeber to pass in the self parameter into the methods, as always.

Python Dictionaries

Dictionaries are used to store data in key:value pairs. They are one of the four data abstractions, meaning they are used to collect data. They are defined using curly braces and they are formated like so-

Dictionaries are ordered, changeable, and do not allow duplicates. That means that items have a defined order that will not change whenever your print or reference the dictionary. You can also replace, add, or remove items in the dictionary after it has been created since they are changeable. Dictionaries also cannot have two items with the same key since duplicates are not allowed. Any duplicates keys will overwrite the exisitng values. Dictionary items can be of any data type, string, integer, float, boolean, etc.

To change values in a dictionary, use keys similiar to how you would use indicies. Use the format dict_name[key] = new_value to change value associated with that key.

update() - use this command can also be used to change values. Pass in the key, value pair as a dictionary into the function to update the dictionary. So format your command as dict_name.update({key:value})

pop() - removes items from the dictionary using a specified key name. Format your command as dict_name.pop(key_name)

popitem() - removes the last inserted item. Format your command as dict_name.popitem()

clear() - dict_name.clear() empties out the dictionary.

del - you can delete an item using by doing del dict_name[key_name]

To access items in a dictionary, use the format dict_name[key_name] to find the value that corresponds to that key. You can also use the get() method in the format dict_name.get(key_name) to get the same result.

len() - len(dict_name) will return the length of the dictionary.

type() - The return value for type(dict) is .

dict() - Pass in another data abstraction into dict() to convert it into a dictionary.

keys() - dict_name.keys() returns a list of all keys in the dictionary.

values() - dict_name.values() returns a list of all the values in the dictionary.

items() - dict_name.items() will return each item or key, value pair in the dictionary as tuples.

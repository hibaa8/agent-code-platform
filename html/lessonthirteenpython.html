Python Functions

Functions are blocks of code that run when called. The benefit of using functions is that if you want to rerun code that you've already written, you wont have to rewrite or copy paste it. Repeating code is considered a bad coding practice and can easily be avoided by using functions. With functions, you can store a block of code in a function and rerun it simply by calling the function.

Python functions are defined with the 'def' keyword followed by the name of the function. After that come any parameters or global variables(variables defined outside of the function) that will be used locally inside of the function. After than comes a colan which defines the start of the body of the function. On the lines below this function, any indented code falls inside of the function. If any variables are defined inside of a function, they have a local scope, meaning they cannot be used outside of the function unless they are returned. You can return a local variable by doing return local_variable_name. However, remember that returning a variable will break you outside of the function so any code that follows the return statement will not be executed, which is why return statements are ussually on the last line of the function. You can also use other keyword like break, continue, and pass inside functions and they have the same functionality mentioned in the loops lesson. Finally, to call a function, simply write function_name() and the code within will be printed.

You can pass in any iterable, variable, or data type as an argument into a function. Arguments are typically passed in when a function is called.There are also different types of parameters
Keyword parameters are set using key=value syntax where the key is the parameter name. These arguments are defined when calling the function and the order in which they are defined does not matter.

Default parameters are when parameters are assigned a default value in the function definition. So, the function can be called without assigning a specific value to that parameter because it already has a default value. But if a value is passed in for that parameter in the function call, then if will overwrite the default value.

Arbitrary arguments can be passed in when you don't know how many parameters will be passed into the function. *args is used when you don't know how many parameters will be passed into the function call. **kwargs are used when you don't know how many keyword arguments will be passed into the function call
